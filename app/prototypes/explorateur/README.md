# Explorateur de données

## Objectif

Reproduire le frame Figma visible pour une nouvelle navigation dans les ressources d'un jeu de données.

## Hypothèse testée

Une navigation latérale par ressources, associée à un aperçu tabulaire immédiat, permet de comprendre plus vite les fichiers disponibles et leur contenu.

## État d'avancement

Prototype haute fidélité statique basé sur le frame Figma sélectionné.

## Inspirations

- data.gouv.fr
- DSFR
- Interfaces d'exploration de tables CSV
