# Test de l’assistant de données

## Objectif

Proposer un parcours partageable qui permet à des utilisateurs de tester l’assistant d’exploration dans le contexte réaliste d’une page de jeu de données data.gouv.fr.

## Hypothèse testée

Un assistant intégré à côté du tableau aide les utilisateurs à comprendre et interroger plus rapidement une ressource tabulaire.

## État d’avancement

Prototype fonctionnel. Le parcours comprend une introduction, le choix d’un jeu de données dont une ressource est publiée ou automatiquement convertie en Parquet, puis l’exploration avec l’assistant existant.

## Inspirations

- Pages de jeux de données data.gouv.fr
- Prototype Explorateur SQL et IA
- Prototype Explore in context
