# Enrichissement de données

## Objectif

Prototyper un parcours d'ajout, d'analyse et d'enrichissement d'une ressource tabulaire sur data.gouv.fr.

## Hypothèse testée

Un utilisateur comprend mieux la qualité et les possibilités d'enrichissement d'un fichier si l'analyse est présentée comme une étape progressive, puis résumée avec un aperçu proche de l'explorateur de données.

## État d'avancement

Première étape en cours : ajout d'une ressource par dépôt de fichier ou lien, puis état d'analyse simulé.

## Inspirations

- Explorateur de données du Prototype Lab
- DSFR pour la hiérarchie, les champs et les boutons
- Parcours Figma "Enrichissement de données"
