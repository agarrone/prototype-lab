# Assistant

## Objectif

Créer un laboratoire isolé pour examiner et améliorer l’interface de l’assistant d’exploration de données.

## Hypothèse testée

Un inventaire interactif des composants, associé à une prévisualisation réaliste, permet d’identifier plus facilement les incohérences de hiérarchie, de densité et de feedback.

## État d’avancement

Première version du catalogue : messages, actions, affichages de données, états système, composer et panneau assistant complet.

Ce prototype est exclusivement visuel. Il ne contient aucun appel au modèle, à DuckDB ou à la route API de l’assistant existant.

## Inspirations

- Storybook et inventaires de composants
- Interfaces d’assistants contextuels
- Principes de divulgation progressive
