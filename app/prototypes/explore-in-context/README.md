# Explore in context

## Objectif

Tester l'intégration de l'explorateur de ressources directement dans une page dataset data.gouv.fr.

## Hypothèse testée

Afficher l'explorateur sous les onglets du jeu de données permet de comprendre le contenu des ressources sans quitter le contexte de la fiche dataset.

## État d'avancement

Prototype en exploration. Le haut de page réutilise le gabarit dataset et l'explorateur est intégré sous la barre d'onglets.

## Inspirations

- Page dataset data.gouv.fr
- Gabarit `templates`
- Prototype `explorateur`
