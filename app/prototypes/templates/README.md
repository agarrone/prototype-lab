# Templates

## Objectif

Créer une bibliothèque de gabarits de pages data.gouv.fr pour tester l'intégration de nouveaux modules d'interface dans des contextes réalistes.

## Hypothèse testée

Un gabarit fidèle de page dataset permet de vérifier plus tôt comment un module comme le nouvel explorateur s'insère dans la hiérarchie, les espacements et les métadonnées existantes de data.gouv.fr.

## État d'avancement

Premier template statique : haut de page du jeu de données "Fichier des personnes décédées", sans header global du site et sans section ressources.

## Inspirations

- data.gouv.fr
- Page `https://www.data.gouv.fr/datasets/fichier-des-personnes-decedees`
