# Recherche

## Objectif

Reproduire la page de recherche avancée des jeux de données de data.gouv.fr.

## Hypothèse testée

Une recherche structurée autour d'une barre principale, de filtres latéraux et de cartes de résultats denses permet de parcourir rapidement le catalogue sans perdre les critères actifs.

## État d'avancement

Prototype haute fidélité statique avec interactions locales pour la recherche, les onglets, les filtres, le tri et la pagination.

## Inspirations

- data.gouv.fr
- DSFR
- Page `https://www.data.gouv.fr/datasets/search`
