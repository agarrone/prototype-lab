# Design System

## General principles

- clarity first
- visual sobriety
- dense but readable
- high fidelity
- production-like realism

## Typography

- Marianne par défaut
- Inconsolata pour le monospace
- Spectral exceptionnellement pour l’éditorial ou le storytelling
- keep typography scale flexible
- use few sizes and strong hierarchy

## Radius

- 4 / 8 / 12
- slightly rounded by default

## Shadows

- light shadows only
- shadows are structural, not decorative

## Borders

- thin borders by default
- structure comes from spacing first

## Colors

- DSFR palette only
- semantic colors from DSFR
- functional colors from DSFR extended palette

## Spacing

- hybrid 4px / 8px system
- 4px for details
- 8px for structure

## Icons

- Remix Icon as default icon system
- standard sizes 16 / 20 / 24
- line icons only
- never use filled icons
- maintain icon consistency across the interface
